Making of portafolio digital:

Aquí te explico el proceso que he seguido para crear este portafolio digital con HTML y CSS:

  1. El primer día elegí como iba a crear la página web, mi elección fue HTML y CSS debido a que es algo que me gusta a pesar de que no sé demasiado. (Hace ya algún tiempo había utilizado HTML y CSS, hasta había hecho un curso básico). Una vez escogida la herramienta con la que iba a desarrollar el portafolio, empecé a hacer la base.

    2. El proceso de creación de la base del portafolio en HTML y CSS fue el siguiente:

    1. Primero cree un logo para la página web basándome en el logo de la NASA (Explicación de la creación del logo de la página).
    2. Generé una paleta de colores para la página web ayudándome de la página web Coolors, esto también fue teniendo en cuenta el color principal del logo de la página. Acto seguido cree la página principal Index.html y el fichero style.css.
    3. Una vez creada la paleta de colores decidí empezar a crear y organizar el menú (tanto el footer como el header).
    4. Para continuar, cree los diferentes documentos .html para poder poner los enlaces a los diferentes apartados en los que se iba a dividir la página.
  2. Una vez creada la base de la página web he ido añadiendo toda la información a cada uno de los apartados. En mi caso primero he añadido la información al apartado de presentación, seguido de IPOP, la página de inicio y los proyectos realizados. Por último. añadí el curriculum que ya tenía hecho del año pasado.
  3. Me he puesto a editar el video, con videos de la página web pexels, audios de YouTube y memes de Youtube (Explicación de la creación del video). Una vez editado el video, lo he subido a YouTube y he copiado el código que proporciona YouTube para añadir videos a páginas web y por último lo he integrado y ajustado a la página web.
  4. Después he estado añadiendo todas las imágenes y dejando bien la página web con el archivo css.
